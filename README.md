# GameWatcher
A tool that automatically tracks the time spent playing games and using apps.

Key Features: “Automatic background measurement,” “Weekly statistical graphs,” “Available in Japanese and English”

Instructions: Download the installer from the “Releases” section and run it.

Virus-checked: https://www.virustotal.com/gui/file/7421b9f0ba944934feabaa9973c36d2c85a1c0af8c1fe1e267d7cd7625c79f9e/detection

This software was developed by an individual and does not have a digital signature, so Windows will display a warning when you download it, but the contents are safe.

How to use: When the program launches, you'll see GameWatcher open in the taskbar. Right-click it and select “Open Statistics.” From the settings menu, you can change the language or choose whether to have it start automatically when Windows boots.
